# Production Worker Lab – Inheritance

**Goal:** Practice inheritance concepts in Java by creating a `ProductionWorker` class that extends the `Employee` class.

### Files
- `Employee.java` — Base class containing employee attributes
- `ProductionWorker.java` — Subclass inheriting from `Employee`
- `WorkerDemo.java` — Main program used to test and demonstrate functionality

### How to Run

Compile:
```bash
javac Employee.java ProductionWorker.java WorkerDemo.java
